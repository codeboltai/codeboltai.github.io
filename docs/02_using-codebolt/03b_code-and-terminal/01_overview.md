# Code & Terminal

The Code and Terminal panels are the primary workspace for editing files, running commands, and reviewing changes. They open automatically when you start a project and remain the foundation of every agent-assisted workflow.

## Opening panels

All panels are accessible from the **Bottom Bar**. Click the panel count badge (e.g. "3 files") to bring that panel into focus, or use the dropdown menus to open a specific panel type.

| Panel | How to open |
|---|---|
| Code editor | Click **Code** badge in the bottom bar, or open any file |
| Terminal | Click **Terminal** badge, or `Ctrl `` ` |
| Git | Click **Git** section in the bottom bar |
| Preview | Debug Tools dropdown → **Preview** |
| Browser | Tools dropdown → **Browser** |
