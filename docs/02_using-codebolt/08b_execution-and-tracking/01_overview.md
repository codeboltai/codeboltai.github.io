# Execution & Tracking

Panels for monitoring long-running work, queued jobs, and the visual Chat Canvas. All accessible from the **Execution** and **Tools** dropdowns in the bottom bar.

For agent-specific monitoring — running processes, thread history, and debug traces — see [Agents](../04_agents/03_running-agents.md). For environment configuration, see [Environments](../08a_environments/01_overview.md).

## Panels in this section

| Panel | Bottom bar path | Purpose |
|---|---|---|
| [Jobs](./05_jobs.md) | Tools → Jobs | Queued and completed background server jobs |
| [Chat Canvas](./06_chat-canvas.md) | Execution → Chat Canvas | Visual canvas for multi-agent coordination |
