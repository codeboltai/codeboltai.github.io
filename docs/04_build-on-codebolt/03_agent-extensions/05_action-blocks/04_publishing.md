# Publishing

Publish from the block root:

```bash
codebolt action actionblock publish --path ./.codebolt/action-blocks/deploy-to-staging
codebolt action actionblock list
```

Publish when the block expresses a stable team-standard operation. Keep it local while the step shape is still moving.

For org-wide distribution, action blocks usually travel inside a [capability](../02_capabilities/01_overview.md).
